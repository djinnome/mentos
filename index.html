mentos.mentos module¶

mentos.mentos.run_mentos(fullS, S, deltaG0, mu0, c_L, c_U, v_L, v_U, initial_log_c, initial_forward_rate, initial_backward_rate, obj, biomass_rxn)[source]¶
run_mentos computes the nonconvex optimization using pyOpt.
\[\begin{split}\underset{x}{\min} & f(x) & \end{split}\]\[\begin{split} & g_j(x) = 0, & j = 1, \ldots, m_e \end{split}\]\[\begin{split} & g_j(x) \leq 0, & j = m_e + 1, \ldots, m \end{split}\]\[\begin{split} & x_{iL} \leq x_i \leq x_{iU}, & i = 1,\ldots, n \end{split}\]

where:

  • x is the vector of design variables
  • f(x) is a nonlinear function
  • g(x) is a linear or nonlinear function
  • n is the number of design variables
  • m_e is the number of equality constraints
  • m is the total number of constraints (number of equality constraints: m_i = m - m_e)

The arguments to this function are as follows

Parameters:
  • fullS (pandas.DataFrame with reactions as columns and metabolites as row index) – The stoichometric matrix data frame object containing all metabolites
  • S (numpy.matrix) – The stoichiometric matrix numpy matrix containing only those metabolites in steady state
  • deltaG0 (numpy.array) – The standard change in Gibbs free energy for all reactions
  • mu0 (numpy.array) – The standard chemical potential numpy array for all metabolites
  • c_L (numpy.array) – The lower bound numpy array on concentrations
  • c_U (numpy.array) – The upper bound numpy array on concentrations
  • v_L (numpy.array) – The lower bound numpy array on fluxes
  • v_U (numpy.array) – The upper bound numpy array on fluxes
  • initial_log_c (numpy.array) – The initial log concentrations (perhaps computed from convex mentos) numpy array
  • initial_forward_rate (numpy.array) – The initial forward rate (perhaps computed from convex mentos) numpy array
  • initial_backward_rate (numpy.array) – The initial backward rate (perhaps computed from convex mentos) numpy array
  • obj (function of 1 argument that returns the value of f(x), g(x), fail) – The objective function. This is a python function of one parameter x. Returns the value of the objective function f, the constraint function g, and a fail flag (0 is pass)
  • biomass_rxn (string) – The name of the biomass reaction in the fullS dataframe.
Returns:

reports for metabolite-sized data and reaction-sized data
Return type:

tuple of pandas.DataFrame
